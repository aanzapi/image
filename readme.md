# Image Hosting Service

Professional image hosting service built with Node.js, Express, and Supabase.

## Features

- 🚀 Fast and reliable image hosting
- 📱 Responsive and modern UI
- 🖼️ Drag & drop upload
- 🔍 Gallery with search functionality
- 📊 Admin dashboard with analytics
- 🗑️ Image management (delete)
- 🔒 Secure file validation
- ⚡ CDN-like delivery

## Tech Stack

- **Backend**: Node.js, Express
- **Database**: Supabase (PostgreSQL)
- **Storage**: Supabase Storage
- **View Engine**: EJS
- **Styling**: Tailwind CSS
- **File Upload**: Multer
- **Security**: Helmet, CORS, Rate Limiting

## Prerequisites

- Node.js 16+
- Supabase account
- npm or yarn
